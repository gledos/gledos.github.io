## 详细参数

由于没有标准化组织/社团进行定义，所以相关的定义一目了然，所见即所得。

+ 必须遵守
  + 使用限定的长度与宽度
+ 常见遵守
  + 图片的格式使用 PNG 或 GIF
  + 使用高度为 5px 的免费字体[Silkscreen](https://web.archive.org/web/20200303050007/https://kottke.org/plus/type/silkscreen/)
  + 字母全部使用大写
  + 不使用透明度通道
